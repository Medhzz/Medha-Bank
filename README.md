# Medha Bank

Medha Bank is a full-stack Bank Management System developed using React, Vite, Firebase, Material UI, and Tailwind CSS.

The project includes:
- Customer Portal
- Admin Portal
- Firebase Authentication
- Firestore Database
- Modern Banking UI

---

# Technologies Used

- React 18
- Vite
- Material UI v5
- Tailwind CSS
- Firebase Authentication
- Firebase Firestore
- React Router DOM
- EmailJS

---

# Features

- Customer account management
- Admin dashboard
- Secure login system
- Firebase authentication
- Firestore database integration
- Responsive UI design
- Protected routes
- Banking dashboard interface

# Installation

Install all dependencies:

```bash
npm install
```

---

# Environment Setup

Rename:

```bash
.env.example
```

to:

```bash
.env
```

Add your Firebase configuration inside the `.env` file.

---

# Run The Project

Start development server:

```bash
npm run dev
```

Open in browser:

```text
http://localhost:5173
```
